# big_quant

## 一、使用方式

### 1. 安装依赖

#### 1.1 （近MacOS用户需要）安装额外依赖

```shell
pip install -r req-macos.txt
```

#### 1.2 安装依赖

```shell
pip install -r req.txt
```

### 2. 填写配置

#### 2.1 填写账户信息`config/accounts`

#### 2.2 填写自选信息`config/contracts`

### 3. 运行主程序

```shell
python big_quant.py
```
